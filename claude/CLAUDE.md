# Conventions

## `,` 始まりのディレクトリ（プロジェクトごとのメモ置き場）

各プロジェクトの root にある `,` 始まりのディレクトリは、私(ken)個人の作業スペース。
global gitignore (`,*/`) で ignore 済み。Claude Code はこれらを前提として把握しておくこと。

- `,` … ちょっとした ignore の書き捨てスクリプトやチェック用ファイルの置き場（throwaway/scratch）
- `,links` … 他プロジェクトを参照用に置いている（シンボリックリンク等）。実装の参考・横断参照に使える
- `,docs` … メモ・ドキュメント置き場

活用方針:
- `,links` の中身は他プロジェクトの参照なので、書き方や実装パターンを調べる材料として読んでよい
- `,` 配下は書き捨て前提。ここのスクリプトやメモを正式な成果物と混同しない
- これらを git にコミットしない（ignore 済み）
